A powerful university and program management plugin with CRM integration, customizable colors, and advanced search capabilities.

== Description ==

**SIT Connect** is the ultimate solution for educational institutions, study abroad agencies, and university portals. Manage universities, programs, campuses, and student applications with ease.

= Key Features =

* **University Management** - Add and manage unlimited universities
* **Program Catalog** - Create detailed program listings with all information
* **Campus Management** - Manage multiple campuses per university
* **Zoho CRM Integration** - Seamless two-way sync with Zoho CRM
* **Customizable Colors** - Match your brand with easy color customization
* **Advanced Search** - Powerful search functionality for students
* **Responsive Design** - Works perfectly on all devices
* **SEO Optimized** - Built-in SEO features for better rankings
* **Multiple Taxonomies** - Countries, faculties, specialities, degrees, languages
* **Lead Management** - Capture and manage student applications
* **Professional Dashboard** - Beautiful admin interface with statistics

== Installation ==

1. Upload the `sit-connect` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to SIT Connect → Settings to configure colors and license
4. Start adding universities and programs!

= Minimum Requirements =

* WordPress 5.2 or greater
* PHP version 7.2 or greater
* MySQL version 5.6 or greater

== Frequently Asked Questions ==

= Do I need a license to use this plugin? =

Yes, SIT Connect requires an active license for full functionality and updates.

= Can I customize the colors? =

Absolutely! Go to SIT Connect → Settings → Color Customization to change all colors to match your brand.

= Does it work with Zoho CRM? =

Yes! SIT Connect has built-in Zoho CRM integration with automatic synchronization.

= Is it translation ready? =

Yes, the plugin is fully translation ready and uses WordPress standards.

= Can I use it with any theme? =

Yes, SIT Connect works with any properly coded WordPress theme.

= Does it work on mobile devices? =

Yes, all features are fully responsive and work perfectly on mobile devices.
